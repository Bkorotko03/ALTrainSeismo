# ALTrainSeismo
Code for our Loyola PHYS 338 SP 2026 (Advanced Lab) semester long project. We hope to characterize the seismic noice induced by Chicago light rail using Arduino to construct a PSD. 
## Structure
Read the folder names fr

## Operation
Flash the Arduino
Stick the experiment to the Arduino
Giver her power
4 single beeps to set up and attempt GPS fix
1 double beep after to confirm GPS fix and recording start
Let it ride
